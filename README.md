# ⚽ CHUTA AÍ! - Sistema de Bolões Esportivos

O **CHUTA AÍ!** é uma plataforma web interativa desenvolvida para o gerenciamento de apostas e palpites em partidas de futebol. O sistema automatiza a criação de ligas, cálculo de pontuações e geração de rankings em tempo real, eliminando a necessidade de controles manuais em planilhas.

## 🚀 Principais Funcionalidades
* **Autenticação Segura:** Login e cadastro com criptografia de senhas utilizando o algoritmo BCrypt.
* **Ligas Privadas:** Criação de bolões com códigos de acesso exclusivos para competições entre amigos.
* **Motor de Palpites:** Inserção e edição de palpites com travas de segurança rigorosas baseadas no horário de início oficial de cada partida.
* **Rankings Dinâmicos:** Classificação de usuários atualizada em tempo real (Global e Privado), com pódio destacado.
* **Painel Gestor (Admin):** Área restrita para o lançamento de partidas, resultados oficiais e moderação de usuários.
* **Dark Mode Premium:** Interface de usuário moderna, totalmente estilizada em tons escuros e detalhes em vermelho para melhor imersão.

## 🛠️ Tecnologias Utilizadas
**Back-end:**
* **Java 23**
* **JSF (JavaServer Faces)** para controle de rotas e regra de negócio.
* **JPA / Hibernate** para mapeamento objeto-relacional (ORM).
* **MySQL 8** como banco de dados relacional.

**Front-end:**
* **PrimeFaces 12** para componentes de interface ricos e responsivos.
* **CSS3 Customizado** (Dark Mode).
